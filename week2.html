Welcome

Hello and welcome to my little corner of the internet. i will be using this page to learn a little coding. I'm currently embarking on a #100DaysOfCode journey and I will be documenting this on here as well as use this as a practice piece.

With this being a practice piece it will likely go through quiete a few changes over time so I hope you can bare with me.

the urrent plan is to complete a Full stack developer course I'm doing on CodeCademy (which I do not count to the time for the 100 day challenge) and on the side practice what I learn on this block as well as on a a little project for tracking details about Sims in The Sims 4™. I know this might strike people as an odd choice of project but it is a franchise that has been close to my heart since it first released and I feel that there is a lot potential for me to practice a large number of technologies on it. Since I need to start with some layout work on that first I am starting my practice in this little blog.

For the moment this will only be about my coding journey, but I will see if I add more other content to it eventually.

Today's Progress: CSS grid and relative sizing for responsive designs

Thoughts: Diving deeper into CSS grid was exciting. I think the possibilities of combining grid and flexbox could be rather interesting.
Maybe I'll finally be able to get a better handle on making sites more responsive, that has previous been difficult for me. I pushed through more of the course than I planned but that cost me the time to work on the Landing page practice, but I will dive into that first thing tomorrow.
One great take away for me from today was the difference between 'em' and 'rem'. I knew there were options for sizing, but not quite got their difference. In the end it's not that hard as the R in 'rem' identifies it as relating to the root element. But then I also mistakenigly thought that the root would be the body element and not actually the html element.
