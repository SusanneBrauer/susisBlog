Day 13: 26th March 2022

Todays plan:

  1. Course work: Start coding for Company Site project ☑
  2. Update journal ☑

Today's Progress: Most complex practice project yet

Thoughts: I made a start setting up the html and styling for my company site project using Flexbox and CSS Grid. I will add responsiveness in the next step. I'm very happy so far about how smoothly layout works with CSS grid. I did mainly use rules related to columns for this but it felt like the right thing to do for this project. I will say that I think I would proably use different pages for this site rather than stacking everything like I have here, but for the purpose of this exercise this will be fine and keep things all in one html file.
At the end I would like ot say that I used my friends studio as practice object. That studio is BlackTree Studio in the UK which is an amazing place for any recording of video or audio, so if you ever need anything like that please check them out. The song used in the "Work Samples" section has been recorded and produced there with me singing.

The picture above shows the current stus of the CC practice project. I'm especially happy with the little blurred edge on the background image in the banner done with an inset box shadow: box-shadow: 0 0 10px 8px #222222 inset;. The colors are bsed on the colors of the actual website which is also where I have taken the pictures and tests from.
The Markings shown in the screen shot are from a great functionality the Firefox Developer Edition offers, which makes working with grid and flexbox even more accessible and controllable. I'm not sure the other browsers offer this but I have only spotted it in there. I think I'll be mainly use this browser to preview sites I work on in the wip phase.
As always I'm tracking my work on this project in my github so if you are intersting at looking at it more closely that is the place to go.
